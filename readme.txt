beamstar -- 2015, 2020 Brenden Deeal
Made with ThreeJS (distributed under the MIT License, see three.js/license for details).
Developed for a Graphical Application Development course at Florida Atlantic University.

Open index.html to run the game.

Tested this in Chrome, if you're using another browser and it doesn't work, use Chrome instead.
Controls are A/S/D jump, arrow-keys move. Dodge the ball and lasers.

files:
* index.html -- main game
* w3r.js -- game source
* three.min.js -- three.js library
* w3r-submission-version.js -- version of the game originally submitted for the course.
